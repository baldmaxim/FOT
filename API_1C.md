# Инструкция: подключение из 1С к FOT Data API

Read-only публичный API для внешних систем. Управление ключами и whitelist таблиц/полей — через админ-вкладку «API-доступ» в FOT.

---

## 1. Базовый URL

- Прод (HTTPS): `https://fotsu10.fvds.ru/external/v1`
- Прод (HTTP): `http://fotsu10.fvds.ru/external/v1`

## 2. Аутентификация

В каждый запрос (кроме `/health`) добавить HTTP-заголовок:

```
Authorization: Bearer fot_<prefix>_<secret>
```

Полный токен выдаётся **один раз** при создании ключа и имеет вид `fot_<16 hex>_<48 hex>`. Если токен потерян — нужно создать новый ключ.

## 3. Доступные endpoints (только GET, read-only)

| Метод | URL | Назначение |
|---|---|---|
| GET | `/external/v1/health` | Проверка доступности (без авторизации) |
| GET | `/external/v1/tables` | Список таблиц, разрешённых ключу |
| GET | `/external/v1/tables/{name}/schema` | Список полей таблицы |
| GET | `/external/v1/tables/{name}` | Чтение строк таблицы |

## 4. Параметры query-string для чтения данных

- `limit` — 1..1000, по умолчанию 100
- `offset` — пропустить N строк, по умолчанию 0
- `order=поле.asc` или `order=поле.desc`
- Фильтры: `eq.поле=значение`, `neq.`, `gt.`, `gte.`, `lt.`, `lte.`, `like.`, `ilike.`, `in.поле=v1,v2,v3`

Поля в фильтрах должны быть в whitelist ключа, иначе 400.

## 5. Формат ответа

```json
{
  "data": [ { ... }, { ... } ],
  "limit": 100,
  "offset": 0
}
```

## 6. Лимиты и ошибки

- Rate limit per-key — настраивается на ключе (default 60 req/min). При превышении — `429 Too Many Requests`.
- `401` — нет/неверный/отозванный/просроченный ключ.
- `404` — таблица недоступна этому ключу.
- `400` — поле не в whitelist, кривые параметры.

## 7. Документация Swagger

`https://fotsu10.fvds.ru/external/v1/docs` — интерактивная документация со всеми endpoints.

## Готовый код для 1С

### Универсальная функция запроса

```bsl
// Возвращает структуру { Код, Тело } — Тело уже распаршен из JSON.
Функция ВыполнитьЗапросFOT(Путь, ПараметрыЗапроса = Неопределено) Экспорт

    Хост      = "fotsu10.fvds.ru";
    Порт      = 443;             // стандартный HTTPS-порт
    ApiKey    = "fot_XXXXXXXXXXXXXXXX_YYYYYYYYYYYYYYYYYYYYYYYYYYYYYYYYYYYYYYYYYYYYYYYY"; // подставить выданный токен

    SSL = Новый ЗащищённоеСоединениеOpenSSL();
    Соединение = Новый HTTPСоединение(Хост, Порт, , , , 30, SSL);

    ПолныйПуть = "/external/v1/" + Путь;
    Если ПараметрыЗапроса <> Неопределено И ПараметрыЗапроса.Количество() > 0 Тогда
        ЧастиQS = Новый Массив;
        Для Каждого КЗ Из ПараметрыЗапроса Цикл
            ЧастиQS.Добавить(КЗ.Ключ + "=" + Строка(КЗ.Значение));
        КонецЦикла;
        ПолныйПуть = ПолныйПуть + "?" + СтрСоединить(ЧастиQS, "&");
    КонецЕсли;

    Заголовки = Новый Соответствие;
    Заголовки.Вставить("Authorization", "Bearer " + ApiKey);
    Заголовки.Вставить("Accept", "application/json");

    Запрос = Новый HTTPЗапрос(ПолныйПуть, Заголовки);
    Ответ  = Соединение.Получить(Запрос);

    Результат = Новый Структура("Код, Тело");
    Результат.Код  = Ответ.КодСостояния;
    ТекстОтвета    = Ответ.ПолучитьТелоКакСтроку("UTF-8");

    Если ПустаяСтрока(ТекстОтвета) Тогда
        Результат.Тело = Неопределено;
    Иначе
        Чтение = Новый ЧтениеJSON;
        Чтение.УстановитьСтроку(ТекстОтвета);
        Результат.Тело = ПрочитатьJSON(Чтение, Истина);
    КонецЕсли;

    Возврат Результат;

КонецФункции
```

### Примеры использования

**Список доступных таблиц:**

```bsl
Ответ = ВыполнитьЗапросFOT("tables");
Если Ответ.Код = 200 Тогда
    Для Каждого Таблица Из Ответ.Тело["data"] Цикл
        Сообщить(Таблица["table_name"]);
    КонецЦикла;
КонецЕсли;
```

**Прочитать первые 50 сотрудников отдела 5 с зарплатой ≥ 50000:**

```bsl
Параметры = Новый Соответствие;
Параметры.Вставить("limit",        50);
Параметры.Вставить("offset",       0);
Параметры.Вставить("order",        "created_at.desc");
Параметры.Вставить("eq.department_id", 5);
Параметры.Вставить("gte.salary",   50000);

Ответ = ВыполнитьЗапросFOT("tables/employees", Параметры);
Для Каждого Стр Из Ответ.Тело["data"] Цикл
    Сообщить(Стр["full_name"]);
КонецЦикла;
```

**Постраничная выгрузка всей таблицы:**

```bsl
Смещение = 0;
Размер   = 1000;
Пока Истина Цикл
    П = Новый Соответствие;
    П.Вставить("limit",  Размер);
    П.Вставить("offset", Смещение);
    Ответ = ВыполнитьЗапросFOT("tables/employees", П);
    Если Ответ.Код <> 200 Или Ответ.Тело["data"].Количество() = 0 Тогда
        Прервать;
    КонецЕсли;
    // ... обработка строк ...
    Смещение = Смещение + Размер;
КонецЦикла;
```

### Если 1С на старом релизе (без OpenSSL по умолчанию)

Использовать HTTP без шифрования: `Хост = "fotsu10.fvds.ru"`, `Порт = 80`, без `ЗащищённоеСоединениеOpenSSL`. Но токен будет летать в открытом виде — рекомендуется только для тестов.

## Что отдать разработчику 1С

1. Базовый URL и API-ключ (через защищённый канал — Telegram secret chat / 1Password / личная встреча).
2. Список таблиц и полей, к которым ему открыт доступ (видно во вкладке «API-доступ» → ключ → «Таблицы»).
3. Эту инструкцию + ссылку на Swagger UI: `https://fotsu10.fvds.ru/external/v1/docs`.

## Проверка работоспособности

```bash
# 1. Health (без ключа)
curl https://fotsu10.fvds.ru/external/v1/health
# → {"ok":true}

# 2. Список таблиц с ключом
curl -H "Authorization: Bearer fot_..." \
  https://fotsu10.fvds.ru/external/v1/tables

# 3. Чтение
curl -H "Authorization: Bearer fot_..." \
  "https://fotsu10.fvds.ru/external/v1/tables/employees?limit=5"
```
